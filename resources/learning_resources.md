# Learning Resources

---

## Free Labs & Practice

**[PortSwigger Web Security Academy](https://portswigger.net/web-security)** — Best free web security labs, covers every major vuln with hands-on exercises.

**[HackTheBox](https://hackthebox.com)** — CTF-style machines, active community, great for practical skills.

**[TryHackMe](https://tryhackme.com)** — Beginner-friendly guided rooms, step-by-step learning paths.

**[PentesterLab](https://pentesterlab.com)** — Web exploitation exercises, many free, excellent quality.

**[OWASP WebGoat](https://owasp.org)** — Deliberately vulnerable web application for learning.

**[DVWA](https://dvwa.co.uk)** — Damn Vulnerable Web App, classic training target.

**[VulnHub](https://vulnhub.com)** — Downloadable vulnerable VMs for offline practice.

**[Hacker101](https://hacker101.com)** — HackerOne's free bug bounty training platform with CTF challenges.

## Key Books

**The Web Application Hacker's Handbook** — Stuttard, Pinto — The definitive reference for web app testing.

**Bug Bounty Bootcamp** — Vickie Li — Purpose-built for bug bounty hunters, practical and modern.

**Real-World Bug Hunting** — Peter Yaworski — Case studies from real disclosed vulnerabilities.

**The Hacker Playbook 3** — Peter Kim — Red team tactics with practical lab exercises.

## Key Talks to Watch

**Jason Haddix — "The Bug Hunter's Methodology"** (DEF CON series, multiple years)
Start with the most recent version — covers recon, tooling, and mindset.

**R-s0n — "Bug Bounty Village Workshop"** (DEF CON 32)
The Ebb & Flow model, attack surface classification, modern recon approach.

**James Kettle — "HTTP Request Smuggling Reborn"** (BlackHat 2019)
Best technical talk on HTTP smuggling — watch before touching the topic.

**Frans Rosen — "Hacking OAuth"** — Deep dive into OAuth token theft chains.

**Martin Vigo — "Attacking Password Reset Flows"** — Every password reset flaw catalogued.
